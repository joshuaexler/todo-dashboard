Todo Daily Dashboard
Generate a bookmark below to get started


Visualize your daily list of tasks in an interactive dashboard.

● Generates using the Todoist Javascript API
● Desktop and mobile friendly
● Syncs with Todoist every sixty seconds

To find your Todoist API token, go to Settings --> Integrations. Copy and paste your API token from the bottom of that page into the text field below.
